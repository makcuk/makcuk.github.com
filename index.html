Nimbus key features

  • Friendly opensource license
  • Designed to store real huge amount of data
  • Distributed hash table based
  • O(1) during data lookups
  • Encrypted storage with certificate-only auth and transport encryption
  • Encrypted metadata
  • Serverless architecture, automatic cloud topology discovery and control
  • No separate metadata and object storage servers
  • Fully redundant storage without using RAID technology
